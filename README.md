# Spring Guide

# 프로젝트 실행 방법
```bash
$ git clone git@github.com:cheese10yun/spring-guide.git
$ gradle clean build
$ gradle bootRun

swagger : http://localhost:8899/swagger-ui/index.html
```

# 구성
```bash
- Java 17
- Spring Boot 3.1.5
- Spring Data JPA
- QueryDSL
- Spring Web
- H2 Database, MySQL Driver
- Lombok

```
